# reinput-shared-workflows
A centralized repository for reusable GitHub Actions workflows, designed to streamline CI/CD processes and standardize pipelines across projects.

## Structure

```
reinput-shared-workflows/
├── .github/
│   └── workflows/
│       ├── reusable-ci.yml
│       └── reusable-cd.yml
└── templates/
    └── Dockerfile.gradle 
```
